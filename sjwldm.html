🧠 神经网络前端代码 · 知识导图
一、代码结构
• 存放位置：HTML 文件内部 <script> 标签
• 代码范围：从注释 // ------------- AI 神经网络模型 开始至文件末尾
• 依赖框架：TensorFlow.js (tf)
• 快速定位关键词：buildModel、trainModelWithCustomLoss、predictNextPosition
二、核心函数
1. 模型构建：buildModel()
2. 模型初始化：initModel()
3. 样本收集：collectSamplesFromRealTraj()、updateTrainingSamplesFromReal()
4. 模型训练：trainModel()、trainModelWithCustomLoss()
5. 模型推理：predictNextPosition()
6. 辅助逻辑：损失函数选择、物理约束计算
三、函数功能
1. buildModel()：定义神经网络结构、编译模型
2. initModel()：初始化/重置模型，释放内存，标记就绪状态
3. collectSamplesFromRealTraj()：从真实轨迹提取训练样本（输入+目标）
4. updateTrainingSamplesFromReal()：维护全局样本库（上限 500 条）
5. trainModel()：训练入口函数，响应用户按钮点击
6. trainModelWithCustomLoss()：自定义训练循环 + 物理约束损失计算
7. predictNextPosition()：实时预测下一时刻位置，生成 AI 轨迹
四、关键实现细节
🔹 网络结构（buildModel）
• 输入层：3 个神经元（归一化前两帧位置 + 当前速度）
• 隐藏层：64 神经元 + ReLU + Dropout(0.2)
• 输出层：1 个神经元（预测下一位置，linear 激活）
• 优化器：Adam(0.003)，损失函数：MSE
🔹 训练细节（trainModelWithCustomLoss）
• 输入：xsTensor、目标：ysTensor、物理理论值：yPhysicsTensor
• 训练方式：optimizer.minimize() 手动梯度下降
• 损失：MSE 损失 + λ×物理约束损失（可切换）
• 触发：每轮训练后自动更新界面损失曲线
🔹 推理细节（predictNextPosition）
• 调用频率：动画循环每 50ms 一次
• 流程：输入归一化 → 张量预测 → 反归一化 → 范围限制
• 兜底：模型未就绪时回退到物理模型
🔹 可调参数
• 网络结构：修改 buildModel()
• 物理权重 λ：修改 trainModelWithCustomLoss() 内 lambda
